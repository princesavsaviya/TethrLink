Why TetherLink

Everything a second monitor needs.
Nothing it doesn't.

Built for Linux power users who want a reliable wired extension display without buying new hardware.

🔌
Zero-config USB transport

Connects over the private 192.168.42.x USB tethering subnet. Plug in, flip USB Tethering on, done.

🖼️
True virtual display

Uses the Mutter ScreenCast D-Bus API to create an independent virtual screen — not a mirror, a real second display.

⚡
Dual codec streaming

Switch between JPEG and H.264 at runtime. Change FPS and quality live — no restart, no reconnect needed.

🔍
Auto-discovery

UDP broadcast means the Android app finds your server automatically. No IP addresses to type or remember.

🎨
GTK4 desktop app

Native Libadwaita UI with a system tray icon, live stats, display arrangement canvas, and persistent settings.

📦
Single .deb install

One apt install command, a desktop entry, and a system tray icon. Installs cleanly, uninstalls cleanly.

How it works

Three steps to a second screen.

No drivers to install, no network configuration, no account required.

Install on Linux

Install the .deb package on your Linux PC. TetherLink appears in your app launcher and system tray.

sudo dpkg -i python3-tetherlink_1.0.0-1_all.deb
Enable USB Tethering

Connect your tablet via USB cable and turn on USB Tethering in Android Settings. The private subnet activates instantly.

Settings → Hotspot & Tethering → USB Tethering
Open the app & connect

Launch TetherLink on your PC and press Start Server. Open the app on your tablet — it discovers and connects automatically.

tetherlink # or click the system tray icon
